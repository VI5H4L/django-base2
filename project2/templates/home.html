Things Learnt

  • CBVs: Simplify CRUD operations with CreateView, UpdateView, DeleteView, and FormView.
  • Form Rendering: Use form.as_p, form.as_table, or form.as_ul to style form fields in templates.
  • Migrations: Fake migrations mark a migration as applied without executing it.

  • Django ORM: Perform CRUD operations, filter data using Q objects, and calculate aggregates like Avg and Sum.
  • Example: Automatically save product names in uppercase using UppercaseCharField.
